# Linda Hamed — Portfolio Site

## Quick Start

```bash
cd linda-hamed-site
npx serve .
```
Then open [http://localhost:3000](http://localhost:3000)

No build step, no dependencies. Pure HTML/CSS/JS.

## Files

| File | Description |
|------|-------------|
| `index.html` | Homepage |
| `brand-guidelines.html` | Living brand reference |
| `CLAUDE.md` | Full design system brief for Claude Code |

## Working with Claude Code

Open this folder in your terminal, then run `claude`.
Claude Code will read `CLAUDE.md` automatically and have full context on the brand system, tokens, animation patterns, and what's still pending.

Good first prompts:
- *"Build the Omantel case study page following the brand system in CLAUDE.md"*
- *"Add a writing/articles index page"*
- *"Make index.html responsive for mobile"*
- *"Replace the LH placeholder logo with [attached image]"*
